70° / 90° / 110° の3角度バリアント × 各2台 = 計6台。スプラウト型Path2。embed 50μm補正済。

実験目的: 内角(W2)の角度がスプラウト軸索のPath2進入率に与える影響を検証
仮説: W2→W1方向に伸びる軸索のスプラウト偏向角 = 内角。小さい角度ほどスプラウトしやすい。
・70° → 偏向70°(容易)、W1軸索は110°でブロック → W2選択性あり
・90° → 偏向90°(臨界閾値) → 対照群
・110° → 偏向110°(困難)、W1軸索は70°で通過 → W1選択性あり(逆転)
設計変更 (v5):
・Path2はスプラウト型: W2から直接出発せず、Path1上のW2から100μm地点でT字分岐
・3角度バリアント: 同一トレイに70°/90°/110°を各2台配置
・W3座標は角度により変化: W3 = (-D×sinθ, +D×cosθ), D=3500μm

設計ポイント

1. 3角度比較実験: 70°/90°/110°で軸索スプラウトの角度依存性を検証。臨界角(~90°)を挟んで上下に配置。
2. スプラウト型Path2: Path2はW2から直接出発せず、Path1上のW2から100μm地点でT字分岐。 W2→W1方向の軸索が分岐点で側枝を出してPath2に進入する想定。
3. embed 50μm補正: 全突起は土台に50μm食い込む。造形高 = 実効突出 + 50μm。
4. ウェル頂部がPDMS面から100μm露出: ウェル実効500μm > リム実効400μm → PDMS硬化後の貫通穴。
5. テーパー/ガイド = 350μm造形(実効300μm): PDMS水面(400μm)より100μm低い。段差なく針が通過。
6. シール＝軸索経路 = 220μm造形(実効170μm): 針φ200μm対応。両者同一高で針が通る。
7. 空気穴6箇所 / リム2mm / 土台5mm: hari200um準拠。3Dプリント再現性・強度・剛性を確保。
8. 角度依存性は仮説段階: T字分岐でのスプラウト角度依存性の直接エビデンスなし。本実験で検証。
